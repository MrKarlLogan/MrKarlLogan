##

Привет!

Меня зовут Игорь, и я фронтенд-разработчик с опытом работы в full-stack разработке. Я разработал более 15 проектов, от клиентских SPA до серверных сервисов.

На фронтенде я в основном работаю с React в сочетании с Redux Toolkit или MobX, а также имею опыт работы с Vue. Моя работа включает в себя игровые SPA на чистом JavaScript с ИИ-противником, командные проекты на React с архитектурой FSD, а также full-stack приложения на React/Vue и Nest.js. На бэкенде я пишу на Node.js (Express, Nest.js) и работаю с базами данных (MongoDB, PostgreSQL, SQLite).
